# Project Overview 
A lightweight order management system showcasing structured logic, clean UI, and real-time calculations.

## Features
 * Add items logic
 * Edit and delete items logic
 * Calculating subtotals with tax
 * Chart dashboard 
 * Generate a receipt 
 * Mobile responsiveness
